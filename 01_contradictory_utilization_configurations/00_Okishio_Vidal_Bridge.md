# Okishio–Vidal Bridge

## Core claim

Okishio establishes that decentralized accumulation cannot produce smooth proportional reproduction.

Vidal establishes that the firm cannot compensate for this through unified planning.

Contradictory utilization configurations are the institutional settlements through which this double failure is managed.

## Role in Chapter 2

This note prevents the theoretical object from collapsing into a labor-process-only argument.

Okishio is constitutive because he provides the market-anarchy side of the contradiction.

## Link to θ

The transformation elasticity θ is the empirical hinge through which these configurations shape productive capacity.
